#NG2-Simple

NG2-Simple is a "walking skeleton" project build to discover Angular2.* with TypeScript.

##You need

    (sudo) npm install -g jspm@beta
    (sudo) npm install tsd -g

##Install

    git clone https://github.com/k33g/ng2-simple.git
    
    npm install
    cd public
    jspm install
    tsd install

##Run

    node app.js
